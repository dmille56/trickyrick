# Installation

## Stable release

To install trickyrick, run this command in your terminal:

```sh
uv add trickyrick
```

Or if you prefer to use `pip`:

```sh
pip install trickyrick
```

## From source

The source files for trickyrick can be downloaded from the [Github repo](https://github.com/dmille56/trickyrick).

You can either clone the public repository:

```sh
git clone git://github.com/dmille56/trickyrick
```

Or download the [tarball](https://github.com/dmille56/trickyrick/tarball/master):

```sh
curl -OJL https://github.com/dmille56/trickyrick/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd trickyrick
uv pip install .
```
